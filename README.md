# 📜 Site Cordel Moderno

Este projeto é uma página web responsiva que apresenta o poema "Cordel Moderno" de Milton Duarte, desenvolvida com o objetivo de aplicar conceitos fundamentais de Front-End, com foco em estruturação semântica e efeitos visuais.

## 🎯 Objetivo do Projeto

Proporcionar uma experiência de leitura imersiva e agradável, unindo a literatura clássica de cordel à tecnologia moderna através de um layout limpo e de um efeito de rolagem dinâmica.

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica utilizando tags como `<header>`, `<main>`, `<section>` e `<footer>`.
* **CSS3:** Design responsivo, estilização de blocos de texto e implementação do efeito Parallax nas imagens de fundo.

## ⚖️ Análise Estratégica & Aprendizagem

Durante o desenvolvimento, foquei-me em três pilares principais:

* **Efeito Parallax:** Utilização da propriedade `background-attachment: fixed` em seções específicas para criar uma sensação de profundidade e dinamismo enquanto o usuário rola a página.
* **Tipografia Independente (Nativa):** Uso estratégico de fontes do próprio sistema operacional (como *Georgia* e *Impact*) e do estilo `italic` para simular a estética de folhetos impressos clássicos, otimizando o carregamento da página sem depender de bibliotecas externas.
* **Design Responsivo e Box Model:** Aplicação de unidades relativas (como `vw` e `vh`), limitação de largura (`max-width`) e alinhamentos automáticos (`margin: auto`) para garantir que o texto não "vaze" e se adapte perfeitamente a computadores e dispositivos móveis.
